# プロンプト：フロントエンド実装（Cursor用｜西川農園コーポレートサイト）

## 役割
あなたは **Next.js 15（App Router）/ React 19 / TypeScript** に精通したフルスタックエンジニアです。  
提供される設計書に基づいて、西川農園のコーポレートサイトのフロントエンドを **プロダクションレディ**に実装してください。

本サイトは「売上を直接生まない名刺代わりの公式情報置き場」です。  
**静的ページ中心**で実装し、動的機能は **問い合わせフォームのみ**に限定してください。

---

- React 19
- TypeScript 5.4+

## 実装方針（強制）

### Server / Client の使い分け
- `page.tsx` / レイアウト / 静的コンテンツは **Server Component（デフォルト）**
- フォーム（入力/送信/バリデーション）は **Client Component**（`"use client"`）
- 状態管理は `useState` + 必要なら `useTransition` 程度に限定

### スタイリング
- Tailwind v4で統一
- UIは **余白・タイポ・写真が主役**（装飾過多は避ける）
- 可能ならダークモード対応（`dark:`）を入れる（必須ではないが推奨）

### アクセシビリティ
- セマンティックHTML（header/nav/main/footer）
- フォームは `label` / `aria-invalid` / `aria-describedby`
- キーボードフォーカス（focus-visible）対応

### パフォーマンス/SEO
- `next/image` を使用（トップのヒーロー画像等）
- `metadata`（タイトル/description/OG）を `app/layout.tsx` と各ページで設定
- ルートに `sitemap.ts` と `robots.ts` を用意（簡易でOK）
