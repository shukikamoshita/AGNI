# 開発計画 (Development Plan)

- **対象プロジェクト名**: 西川農園 コーポレートサイト
- **参照元**: `detailed_requirements_specification.md` (11. 開発ロードマップ)
- **更新日**: 2026-02-01

## 1. 目的
プロジェクトを円滑に進めるための環境構築手順と実装フェーズを定義する。

## 2. 開発環境のセットアップ
1. **GitHub リポジトリ作成**: `nishikawa-farm-web`
2. **Next.js プロジェクト初期化**: `npx create-next-app@latest .` (App Router, Tailwind, TS含む)
3. **Cursorのセットアップ**: 開発効率化のため。
4. **Supabase プロジェクト作成**: データベースおよびAPIキーの取得。
5. **Vercel 連携**: 自動デプロイの設定。

## 3. 実装フェーズ

### フェーズ1: 基盤構築 (1〜2日)
- プロジェクト初期化
- CSS設計、共通コンポーネント(Header/Footer)作成
- Supabase クライアント設定

### フェーズ2: 各ページ実装 (2〜4日)
- トップページ
- ストーリーページ
- 作物・メンバー紹介ページ

### フェーズ3: お問い合わせ機能 (1日)
- お問い合わせ画面
- Server Actions 実装
- スパム対策、サンクスページ

### フェーズ4: 検証・公開 (1日)
- デバイス実機検証
- SEO/OGP設定
- 本番用ドメイン接続

## 4. 運用・保守
- **コンテンツ更新**: 学生メンバーが担当し、GitHub PR 経由で本番反映。
- **監視**: Vercel のダッシュボードでのエラーログ確認。
