# UI/UXデザイン設計 (UI/UX Design)

- **対象プロジェクト名**: 西川農園 コーポレートサイト
- **参照元**: `detailed_requirements_specification.md` (5. UI/UX設計)
- **更新日**: 2026-02-01

## 1. 目的
ビジュアルおよびユーザー体験の設計方針を定義し、一貫性のあるサイト構築を実現する。

## 2. デザインコンセプト
**「土の温もりと、若き躍動」**
農業への敬意(歴史)と学生の挑戦(未来)が共存する、親しみやすくも洗練された世界観。

## 3. スタイルガイド

### 3.1 カラーパレット
- **Primary**: `#2D5A27` (森の緑 / 信頼、穏やかさ)
- **Secondary**: `#F2E8D5` (土のベージュ / 温かみ、ベースカラー)
- **Accent**: `#B3543D` (テラコッタ / 果実、活動の情熱)
- **Text**: `#333333` (深いグレー / 読みやすさ重視)

### 3.2 タイポグラフィ
- **見出し**: 日本語明朝体 (例: Noto Serif JP) - 落ち着き・伝統。
- **本文**: 日本語ゴシック体 (例: Noto Sans JP) - 明快・モダン。

## 4. ページ構成と要素

### 4.1 トップページ
- **Hero**: 高品質なドローン空撮または農作業のイメージ写真 + ストレートなメッセージ。
- **Introduction**: 農園の3つの特徴をカード型orアイコンで紹介。
- **Story Snapshot**: 農園ストーリーへの入り口。

### 4.2 農園紹介 (ストーリー)
- **構成**: エッセイ風のレイアウト。写真と文章が交互に現れ、読了を促す。

### 4.3 お問い合わせ
- **構成**: シンプルな1カラム。ストレスを与えない設計。

## 5. UIパーツ設計 (仮定)
- **ボタン**: 角丸を控えめにした高級感のあるデザイン。
- **アニメーション**: スクロール時のフェードイン（控えめに設定）。

## 6. レスポンシブ設計
- **モバイルファースト**: ハンバーガーメニュー、シングルカラムのレイアウトを優先して設計。
